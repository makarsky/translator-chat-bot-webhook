Taking translations to a whole new level

This is a Telegram chat bot and it can translate your messages to any of 105 languages

The nice pagination feature was made to make your experience better and faster. 7 recently used languages are always listed on the first page.

Tech stack: Node.js, Express, JavaScript, Redis Telegram API, Translate API, Text-To-Speech API, Heroku

Try it

If you don't have the Telegram app, you can download it using one the following app stores:

Google Play AppStore

Awesome features

You can forward messages from other chats and use text-to-speech (up to 200 characters; not for all languages).

It is completely free!

Attention to details

You don't have to manually switch between 2 recently used languages. The chat bot does it for you automagically

4 interface languages

The bot has only 4 interface languages at the moment. You can become a contributor by adding a new interface language!

The bot uses the unofficial google translate API , so sometimes a translated text may differ from translate.google.com. Generally speaking, translation isn't perfect, but it is ok
